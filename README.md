# 📘 Atividade — Funções em PHP (funcionalidade.php)

## 📌 Descrição

Este projeto consiste na criação de um arquivo `funcionalidade.php` contendo diversas funções em PHP, organizadas em níveis de dificuldade, com o objetivo de praticar conceitos fundamentais de programação.

As funções foram divididas em quatro níveis, abordando diferentes formas de uso:
- Sem retorno e sem parâmetro
- Sem retorno e com parâmetro
- Com retorno e sem parâmetro
- Com retorno e com parâmetro

---

# 🎯 Objetivo

Praticar:
- Criação de funções em PHP
- Uso de parâmetros
- Retorno de valores
- Manipulação de dados
- Estruturas de repetição e condição
- Organização de código
